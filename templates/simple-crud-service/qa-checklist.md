# QA 체크리스트 — Simple CRUD Service

## 기능
- [ ] 생성(Create)이 정상 동작한다
- [ ] 조회(Read)가 정상 동작한다
- [ ] 수정(Update)이 정상 동작한다 (필요한 경우)
- [ ] 삭제(Delete)가 정상 동작한다 (필요한 경우)
- [ ] 빈 입력, 너무 긴 입력에서 안전하게 처리된다

## DB 안전
- [ ] DB 접근 권한이 최소화되어 있다
- [ ] (Supabase) RLS(Row Level Security)가 활성화되어 있다
- [ ] 백업 정책이 정해져 있다 (필요한 경우)

## 보안
- [ ] DB 키 / 접속 정보가 클라이언트 코드에 노출되지 않는다
- [ ] GitHub에 `.env`가 올라가 있지 않다
- [ ] 인증이 필요한 페이지에 인증 없이 접근되지 않는다

## 개인정보
- [ ] `privacy.md`에 실제 수집 항목 / 목적 / 보관 기간이 정확히 적혀있다
- [ ] `privacy.md` 페이지 링크가 있다
- [ ] 자리표시자가 모두 실제 값으로 교체되어 있다
