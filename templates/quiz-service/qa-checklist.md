# QA 체크리스트 — Quiz Service

## 기능
- [ ] 처음부터 끝까지 모든 질문이 정상 동작한다
- [ ] 모든 결과 분기로 끝까지 도달 가능한지 검증한다 (각 결과 케이스 1번씩)
- [ ] 결과 페이지에서 다시 시작이 가능하다

## 결과 공유
- [ ] 결과 URL을 새 창에서 열어도 결과가 동일하게 보인다
- [ ] 카카오톡 / Slack 등에 공유했을 때 OG 이미지 / 제목이 깨지지 않는다
- [ ] 모바일에서 결과 화면이 잘 보인다

## 콘텐츠
- [ ] 오타 없음
- [ ] 결과 텍스트가 부정적 / 차별적 표현을 포함하지 않는다
- [ ] 자리표시자(`[서비스명]` 등)가 모두 실제 값으로 교체되어 있다

## 개인정보
- [ ] 사용자 답변이 식별 가능한 형태로 저장되지 않는다
- [ ] `privacy.md` 페이지 링크가 있다
