# Quiz Service Template

MBTI, 성향 테스트 같은 결과형 서비스 템플릿입니다.

## 적합한 서비스
- 성격 / 성향 테스트
- 추천 결과형 도구 (예: 직무 추천)
- 결과 공유가 핵심인 서비스

## 권장 스택
- 정적 사이트(JSON 기반 결과) 또는 Next.js
- DB는 보통 불필요 (결과를 그 자리에서만 보여줌)
- 통계가 필요하면 Neon

## 사용 절차
1. 이 폴더 내용을 새 GitHub repo로 복사
2. 질문 / 결과 데이터 정리
3. 자리표시자(`[서비스명]`, `[담당자 이메일]`) 교체
4. 배포 → QA → 노출 (각 단계 분리)

## 결과 공유 시 주의
- 공유용 OG 이미지가 모바일 카카오톡 등에서 깨지지 않게 설정
- 결과 URL이 그대로 공유되었을 때 서비스가 정상 동작하는지 확인
