# QA 체크리스트 — AI Service

## 기능
- [ ] AI 호출이 정상 응답한다
- [ ] AI 호출 실패 시 사용자에게 에러 메시지가 보인다 (흰 화면 X)
- [ ] 호출 중 로딩 표시가 있다
- [ ] 빈 입력, 너무 긴 입력에서 안전하게 처리된다

## AI 안내
- [ ] `ai-notice.md` 내용이 사용자가 볼 수 있는 곳에 노출된다
- [ ] AI 응답을 신뢰할 수 없다는 안내 문구가 있다

## 보안
- [ ] API 키가 브라우저(클라이언트 코드)에서 보이지 않는다
- [ ] 키가 GitHub에 올라가 있지 않다 (`.env` 미커밋)
- [ ] 호출 제한(rate limit)이 설정되어 있다

## 비용
- [ ] AI 사업자 대시보드에 사용량 한도(monthly cap)가 설정되어 있다
- [ ] 비정상 호출이 감지되면 사이트를 임시로 내릴 수 있는 절차가 있다

## 개인정보
- [ ] 사용자 입력 원문이 서버 로그에 그대로 남지 않는다
- [ ] `privacy.md` 페이지 링크가 있다
