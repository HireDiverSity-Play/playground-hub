# 개인정보 처리방침 (템플릿) — AI Service

> 이 문서는 템플릿입니다. `[서비스명]`, `[담당자 이메일]`, `[OpenAI / Anthropic 등]` 자리표시자를 실제 값으로 교체한 뒤 사용하세요.

## 1. 수집하는 정보
- 사용자가 입력한 텍스트 / 이미지 (AI 처리에만 사용)

## 2. 외부 제공
- 입력 내용은 `[OpenAI / Anthropic 등]`의 API로 전송되어 처리됩니다.
- 자세한 처리 방식은 해당 외부 사업자의 정책을 따릅니다.

## 3. 보관
- `[서비스명]`은 사용자 입력을 자체 DB에 저장하지 않습니다.
  (또는 통계 / 디버깅 목적으로 `[N일]` 보관 후 파기하는 경우 그 기간을 명시합니다.)

## 4. 사용자 권리
- 사용자는 `[담당자 이메일]`로 본인 정보 처리에 관해 문의할 수 있습니다.

## 5. 처리방침 변경
- 변경 시 본 페이지를 통해 사전에 공지합니다.
