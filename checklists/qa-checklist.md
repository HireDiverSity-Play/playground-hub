# QA 체크리스트

## 기능
- [ ] 메인 흐름이 끝까지 동작한다
- [ ] 빈 입력, 너무 긴 입력에서 멈추지 않는다
- [ ] 에러 발생 시 사용자에게 알린다 (흰 화면 X)
- [ ] 새 시크릿 창에서도 동작한다

## 화면
- [ ] 모바일에서 잘 보인다
- [ ] PC에서 잘 보인다
- [ ] 버튼 / 텍스트가 잘리지 않는다
- [ ] 다크모드 / 라이트모드(있다면) 둘 다 보인다

## 성능
- [ ] 첫 화면이 약 5초 안에 뜬다
- [ ] 외부 호출(AI 등) 시 로딩 표시가 있다

## 콘텐츠
- [ ] 오타 없음
- [ ] 외부 링크가 모두 살아있다
- [ ] 자리표시자(예: [서비스명], [담당자 이메일])가 모두 실제 값으로 교체되어 있다

- [ ] 분석 도구가 개인정보를 무단 수집하지 않는다
