# 개인정보 체크리스트

## 수집
- [ ] 꼭 필요한 정보만 수집한다
- [ ] 수집 항목 / 목적 / 보관 기간이 화면에 표시된다
- [ ] 수집 동의를 사용자가 명시적으로 한다

## 저장
- [ ] 비밀번호 등 민감 정보는 평문으로 저장하지 않는다
- [ ] DB 접근 권한이 최소화되어 있다
- [ ] 백업 / 덤프에 개인정보가 그대로 들어가지 않는다
- [ ] (Supabase) 행 단위 보안(RLS)이 활성화되어 있다

## 전송
- [ ] HTTPS로만 통신한다
- [ ] 외부 서비스(AI, 분석, 광고 등)에 전송되는 데이터를 사용자에게 알린다

## 로그 / 추적
- [ ] 사용자 입력 원문이 로그에 그대로 남지 않는다
- [ ] IP, 식별자가 필요 이상으로 저장되지 않는다

## 처리방침
- [ ] `privacy.md` 또는 동일한 처리방침 페이지가 연결되어 있다
- [ ] 처리방침에 위 항목이 정확히 반영되어 있다
- [ ] 처리방침의 자리표시자(예: [서비스명], [담당자 이메일])가 모두 실제 값으로 교체되어 있다

## 사고 대응
- [ ] 문제가 생겼을 때 누구에게 알릴지 정해져 있다
- [ ] 데이터 삭제 요청을 처리할 방법이 정해져 있다
