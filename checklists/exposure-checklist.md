# 노출 전 체크리스트

이 체크리스트의 모든 항목이 통과해야 메인서비스에 노출할 수 있습니다.

## 배포
- [ ] 배포 URL이 외부에서 접속 가능하다
- [ ] 모바일 브라우저에서 정상 동작한다
- [ ] PC 브라우저에서 정상 동작한다
- [ ] HTTPS로 접속된다 (자물쇠 표시)

## 개인정보
- [ ] [`privacy-checklist.md`](./privacy-checklist.md) 모든 항목 통과
- [ ] 개인정보 처리방침 페이지가 연결되어 있다
- [ ] 이용약관 페이지가 연결되어 있다 (필요한 경우)

## AI (해당 시)
- [ ] 외부 AI 사용 사실이 사용자에게 표시된다
- [ ] `ai-notice.md` 또는 동일한 안내 문구가 화면에 노출된다
- [ ] 비용 한도(monthly cap) 또는 호출 제한이 설정되어 있다

## QA
- [ ] [`qa-checklist.md`](./qa-checklist.md) 모든 항목 통과

## 운영 준비
- [ ] 누가 책임지고 운영하는지 정해져 있다
- [ ] 서비스를 중단해야 할 때 어떻게 내릴지 합의되어 있다
- [ ] 사용자 문의 받을 창구가 정해져 있다

## 메인서비스 반영
- [ ] 메인서비스 repo의 `serviceLinks`에 항목 추가
- [ ] 메인서비스 배포 완료
- [ ] 노출된 메뉴에서 실제로 클릭해 보고 이동까지 확인

> 메인서비스 노출 자체는 playground-hub가 아니라 **메인서비스 repo의 `serviceLinks` 코드에서 관리**합니다. 자세한 내용은 [`docs/exposure-policy.md`](../docs/exposure-policy.md) 참고.
